## Estado actual

### Barra de navegación inferior (`src/components/ui/Layout.tsx`)

**Problema resuelto:** El icono de Bodega no se veía cuando estaba activo.

**Causa raíz:** `#C9A84C` (gold) tiene luminosidad perceptual más baja que `#F0EBE1` (cream) al 45% de opacidad. El tab activo parecía más oscuro que los inactivos.

**Solución aplicada:**
- Color activo cambiado a `#F5C842` (gold más saturado y brillante)
- Opacidad inactiva reducida de `0.45` a `0.35`
- `position: relative; zIndex: 10` añadido al `<nav>` para evitar que el overlay fixed de Bodega lo tape
- Prop `end` en NavLinks para evitar matching parcial de rutas

**Icono Bodega:** SVG de barril con rectángulos rellenos (`fill="currentColor"`):
```tsx
<path d="M6 3h12v2H6zM6 19h12v2H6zM4 5h2v14H4zM18 5h2v14h-2zM6 11h12v2H6z"/>
```
