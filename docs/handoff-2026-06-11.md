### Flujo
```
Recibir Imagen
    ├→ Preparar Mensajes → Analizar Vision → Parsear Resultado → [IF Tiene Nombre]
    │                                                            ↙ TRUE    FALSE ↘
    │                                                   Generar Studio    Buscar Fallback
    │                                                            ↘              ↙
    │                                                     (ambas conectadas a)
    │                                                     Extraer URL Imagen
    │                                                                ↑
    └→ Preparar Trasera → [IF Tiene Trasera]
                              ↙ TRUE      FALSE ↘
                        Recortar Trasera    Sin Trasera
                              ↘                ↙
                           Extraer URL Imagen
                                    ↓
                              Responder
```

`Extraer URL Imagen` usa `$('NodeName').first()` para recoger resultados de todas las ramas.

### Variables de entorno en n8n (configuradas en Portainer)
- `OPENAI_API_KEY` — para "Recortar Trasera" vía `$env`
- `NODE_FUNCTION_ALLOW_BUILTIN=*` — para `require('https')` en Code nodes

### Nodos destacados
- **Generar Imagen Studio**: `gpt-image-1`, `1024x1536`, `quality: medium`
- **Recortar Trasera**: Code node, `POST /v1/images/edits` multipart, PNG transparente
- **Extraer URL Imagen**: devuelve `imagen_url` + `imagen_trasera_url`

### Advertencias (falsos positivos, no bloquean)
- `INVALID_PARAMETER` en "Generar Imagen Studio" y "Analizar Vision"
- `HARDCODED_CREDENTIALS` en "Buscar Imagen Fallback" (SerpAPI key — deuda técnica)
